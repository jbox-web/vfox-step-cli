# vfox-step-cli

[![CI](https://github.com/jbox-web/vfox-step-cli/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-step-cli/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [step-cli](https://github.com/smallstep/cli)

## Install

```
mise plugin install step-cli https://github.com/jbox-web/vfox-step-cli.git
```
